Rate Limits
Global: 60 requests per minute per IP across all /api/* routes.

Individual endpoints may have tighter limits (e.g. explore: 30/min, battle: 60/min).
When exceeded, you receive HTTP 429 with Retry-After: 60.

API key requests are still IP-counted. Use a single key per deployment, not per request.
Global /api/* — 60 req / minute / IP
Explore / Travel / Mine — 30 req / minute
Rest — 20 req / minute
Buy / Sell / Equip / Battle — 60 req / minute
Create API Key — 10 req / minute
Chat send — 20 req / minute

Client Libraries
Python
JavaScript
cURL

Download ol2_client.py and place it next to your script. No external dependencies required.

⬇ Download ol2_client.py
from ol2_client import OL2Client client = OL2Client("ol2_your_key_here") # Read game state state = client.game_state() print(state["player"]["name"], "— Level", state["player"]["level"]) # Explore the world result = client.explore() print(result["message"]) # Battle actions (when in_battle=True) if state["in_battle"]: print(client.battle_attack()) # Chat msgs = client.chat_history(limit=20) client.chat_send("Hello from the API!") # World info areas = client.world_areas() events = client.world_events() # Leaderboard lb = client.leaderboard()

Include ol2_api.js in your HTML or import it as an ES module.

⬇ Download ol2_api.js
import { OL2Client } from './ol2_api.js'; const client = new OL2Client('ol2_your_key_here'); // Read game state const state = await client.gameState(); console.log(state.player.name, 'Level', state.player.level); // Explore const result = await client.explore(); console.log(result.message); // Chat const msgs = await client.chatHistory(20); await client.chatSend('Hello from JS!'); // Leaderboard const lb = await client.leaderboard();
# Set your key export OL2_KEY="ol2_your_key_here" export OL2_BASE="https://{{ host }}" # Get game state curl -s -H "X-API-Key: $OL2_KEY" "$OL2_BASE/api/game/state" | python -m json.tool # List available scopes curl -s "$OL2_BASE/api/keys/scopes" | python -m json.tool # Explore curl -s -X POST -H "X-API-Key: $OL2_KEY" "$OL2_BASE/api/action/explore" | python -m json.tool # Attack (if in battle) curl -s -X POST -H "X-API-Key: $OL2_KEY" "$OL2_BASE/api/battle/attack" | python -m json.tool # Chat history curl -s -H "X-API-Key: $OL2_KEY" "$OL2_BASE/api/social/chat?limit=20" | python -m json.tool # Send chat curl -s -X POST -H "X-API-Key: $OL2_KEY" -H "Content-Type: application/json" \ -d '{"message":"Hello from cURL!"}' "$OL2_BASE/api/social/chat" | python -m json.tool # Leaderboard curl -s "$OL2_BASE/api/social/leaderboard" | python -m json.tool # World events curl -s "$OL2_BASE/api/world/events" | python -m json.tool
